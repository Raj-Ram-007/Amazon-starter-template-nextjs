Get Emmit working 

    Open Command Palette (Ctrl + Shift + P)

    Open Settings (JSON)

    Add to the end:

    "emmet.triggerExpansionOnTab": true,
    "emmet.includeLanguages": {
        "javascript": "javascriptreact",
        "vue-html": "html",
        "razor": "html",
        "plaintext": "jade"
    }
